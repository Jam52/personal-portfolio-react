# Personal-Portfolio

A single page web application to showcase my skills and portfolio.

### Built with

- HTML
- SCSS
- JavaScript
- React

## Installation

### Prerequisites

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/Jam52/personal-portfolio-react.git
```

2. Install NPM packages

```sh
npm install
```

4. Run dev enviroment

```sh
npm run dev
```

4. Build project

```sh
npm run build
```

6. Run project

```sh
npm start
```
